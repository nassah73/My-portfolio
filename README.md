# 🚀 My Personal Portfolio

Welcome to the repository of my personal portfolio! This project showcases my skills, projects, and journey as a **Full-Stack Developer** & **Data Science Enthusiast**.

## ✨ Key Features
* 📱 **Fully Responsive:** Optimized for all screen sizes.
* 🎨 **Modern UI:** Built with a "Mobile-First" approach using **Tailwind CSS**.
* ⚡ **Performance:** Fast loading times and smooth transitions with **Framer Motion**.
* 🛠️ **Modular Code:** Built using reusable **React Components**.
